## Neuer Anlauf: Nur die Liste

Eine Liste hält einfach nur Werte fest – genau wie eine Einkaufsliste oder eine Namensliste.

```python
# Eine einfache Liste von Namen:
mitarbeiter = ["Anna", "Ben", "Chris"]

```

Wenn du jetzt an einen bestimmten Namen heranwillst, musst du wissen, dass Python immer bei **0** anfängt zu zählen:

* `mitarbeiter[0]` ist **Anna**
* `mitarbeiter[1]` ist **Ben**
* `mitarbeiter[2]` ist **Chris**
